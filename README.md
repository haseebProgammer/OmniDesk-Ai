# 🚀 OmniDesk AI - Your Front Desk. Always On.

OmniDesk AI is a premium SaaS platform that provides high-end AI receptionist solutions for local businesses (Dental, Chiropractic, Salons, and Restaurants). Our AI ensures that no business ever loses a customer to a missed call again by providing 24/7 intelligent voice and booking automation.

![OmniDesk AI Hero](/public/hero.png)

## ✨ Features

- **24/7 Intelligent Receptionist**: Handles calls, books appointments, and answers FAQs at any time.
- **Premium 3D Experience**: Cinematic UI built with Three.js and Framer Motion for a high-converting sales experience.
- **Full Multi-Page SaaS Structure**:
  - **Services**: Specialized solutions for medical and service-based industries.
  - **ROI Calculator Guidance**: Step-by-step roadmap for business owners to understand the value of AI.
  - **Dynamic Blog**: Humanized, data-driven articles on industry trends and ROI.
  - **Pricing**: Performance-based plans designed to scale with local businesses.
- **Seamless Communication**: Integrated WhatsApp button and professional contact forms.
- **Glassmorphism Design**: Modern, sci-fi inspired aesthetic using Tailwind CSS 4.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/) (CSS-first configuration)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) & [Lenis](https://lenis.darkroom.engineering/) (Smooth Scroll)
- **3D Elements**: [Three.js](https://threejs.org/) & [@react-three/fiber](https://docs.pmnd.rs/react-three-fiber)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Routing**: [React Router DOM](https://reactrouter.com/)

## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS version recommended)
- npm or yarn

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/haseebProgammer/OmniDesk-Ai.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
